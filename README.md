Um sistema web administrativo para supermercados, desenvolvido como protótipo funcional.
Permite cadastrar, editar e excluir produtos, gerenciar promoções, visualizar usuários, fazer upload de fotos e muito mais.

 Descrição do Projeto

Este projeto consiste em um painel administrativo completo, com autenticação, CRUDs e integração com um backend Node.js + PostgreSQL.

Ele foi desenvolvido utilizando:

React + Vite

Styled-Components

React Router

Fetch API

Node.js + Express (backend)

PostgreSQL (banco de dados)

O objetivo é fornecer uma interface moderna, organizada e simples para gerenciar um supermercado.
